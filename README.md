# Amharic Alphabet Reading App

Interactive Amharic alphabet learning application with audio pronunciation, multi-language transcription support, and vocabulary words.

## Features

- Interactive character grid with color-coded groups
- Audio pronunciation (TTS + real audio files)
- Multi-language transcription (English, Arabic, Hebrew, Oromiffa)
- Vocabulary words with examples
- Cycle mode for practice
- Responsive design

## Quick Deploy to Vercel

1. Visit https://vercel.com
2. Sign in with GitHub (or create account)
3. Click "Add New Project"
4. Import this folder by uploading it as a ZIP or connecting to GitHub
5. Vercel will auto-detect Vite and deploy!

## Local Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

The app will be available at http://localhost:5173

## Technologies Used

- React 18
- Vite
- Tailwind CSS
- Lucide React (icons)
